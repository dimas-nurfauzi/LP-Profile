# Landing Page

A modern and responsive landing page built with HTML, Tailwind CSS, and JavaScript.

## ✨ Features

- **Fully Responsive:** Optimized for all screen sizes (mobile, tablet, desktop)
- **Tailwind CSS:** Utility-first styling for fast and consistent design
- **Interactive Elements:** Smooth animations and dynamic components with vanilla JavaScript
- **Clean UI:** Minimalist and modern design aesthetic

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **Tailwind CSS** – Styling and responsive design
- **JavaScript** – Interactive functionality (animations, DOM manipulation, etc.)
